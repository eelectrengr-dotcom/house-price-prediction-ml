🏠 House Price Prediction App

📌 Description
This is a Machine Learning web application built using Streamlit.  
It predicts house prices based on user inputs like area, number of bedrooms and bathrooms.

---

🚀 Features
- Predict house prices instantly
- User-friendly interface
- Uses Linear Regression and Random Forest models
- Real-time prediction

---

🧠 Models Used
- Linear Regression
- Random Forest Regressor

---

📊 Input Features
- Area (sq ft)
- Bedrooms
- Bathrooms

---

🛠 Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

---
📁 Project Files
- app.py
- data.csv
- README.md

---

▶️ How to Run

Step 1: Install libraries
pip install streamlit pandas scikit-learn

Step 2: Run the app
python -m streamlit run app.py
